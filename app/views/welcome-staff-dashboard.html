Welcome to GOV.UK Pay

This sandbox account allows you to test the features of GOV.UK Pay without taking any real payments.

Get started

Make some test payments that will populate your sandbox account and allow you to test the features of this admin tool.

1. Generate an API key and copy it to your clipboard.

2. Go to make a test payment. You will be asked to insert the API key you have generated plus some pretend payment details.

3. Go to the GOV.UK Pay documentation to see the different test cards you can use.

4. You can then see these payments populate your transactions list.

When you are ready to take real payments get in touch with us.
